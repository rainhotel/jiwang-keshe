# 聊天软件使用说明

## 环境要求

- Python 3.6+
- 标准库即可，无需安装额外依赖（tkinter、socket、json、threading 均为 Python 标准库）

## 启动方式

### 1. 启动服务端

在终端中运行：

```bash
python server.py
```

服务端默认监听 `127.0.0.1:9999`，可以在代码中修改 `DEFAULT_HOST` 和 `DEFAULT_PORT`。

### 2. 启动客户端

在终端中运行：

```bash
python client.py
```

将弹出登录窗口，默认连接 `127.0.0.1:9999`。

## 使用流程

1. **注册**：输入用户名和密码，点击"注册"按钮
2. **登录**：注册成功后，输入用户名和密码，点击"登录"按钮
3. **公聊**：在输入框中输入消息，按回车或点击"发送"按钮，所有人可见
4. **私聊**：
   - 方式一：在输入框中输入 `@用户名 消息内容` 发送
   - 方式二：双击右侧在线用户列表中的用户，输入私聊内容
5. **查看在线用户**：右侧面板显示当前在线用户列表

## 多客户端测试

在同一台机器上打开多个终端，分别运行 `python client.py`，每个客户端可以使用不同用户名登录。

## 消息颜色说明

- 默认颜色：公聊消息
- 蓝色：系统消息（加入/离开通知）
- 绿色：私聊消息
- 红色：错误或断开连接提示

## 用户数据

注册的用户信息保存在 `users.json` 文件中（与 server.py 同目录），服务端重启后数据不丢失。
